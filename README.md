## Project Overview
This project analyzes hotel booking cancellations using Microsoft Excel dashboards and data visualization techniques.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Data Visualization

## Features
- Booking analysis by hotel type
- Cancellation trend analysis
- Guest category analysis
- Monthly booking insights
- Interactive dashboard

## Workbook Contents
1. Raw_Data
2. Pivot_Analysis
3. Dashboard

## Key Insights
- City hotels had higher bookings and cancellations.
- Couples contributed the highest number of bookings.
- September and October showed high booking activity.
- Desired room types had more cancellations.
